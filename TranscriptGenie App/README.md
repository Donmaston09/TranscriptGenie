# 🎙️ TranscriptGenie

Generate and download video transcripts (with timestamps) from:

- ✅ YouTube links  
- ✅ Uploaded local video/audio files  

Built with **Streamlit**, **Whisper**, and **yt-dlp**.

## 📌 Features

- Extract existing YouTube captions (fast method)
- Automatically fallback to Whisper transcription if captions are unavailable
- Upload local video/audio files for transcription
- Timestamped transcript output
- Export transcript as a Word (.docx) file
- Clean and simple Streamlit interface

---

## 🛠️ Tech Stack

- Streamlit  
- OpenAI Whisper  
- youtube-transcript-api  
- yt-dlp  
- python-docx  
- PyTorch
